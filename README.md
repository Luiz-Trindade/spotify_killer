## ✨ Features

<table>
<tr>
<td width="50%">

### 🎬 Video & Audio

Download in **best quality** available — video with audio merged, or audio-only in M4A format.

### 📋 Playlist Support

Paste a single video URL or an entire playlist — all links are extracted automatically.

</td>
<td width="50%">

### ⚡ Parallel Downloads

Uses `ThreadPoolExecutor` to download multiple files simultaneously.

### 📁 Smart Organization

Files are automatically organized by **type** and **date** (DD.MM.YYYY).

</td>
</tr>
</table>

Powered by [Rich](https://github.com/Textualize/rich) with:

- 🟢 Spotify-inspired green theme
- 📊 Progress bars with spinners
- 🎯 Interactive selection menus
- 📋 Summary tables after completion
